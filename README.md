# BO2 Plutonium Trickshot Setup Guide

This repository exists to help simplify installation for BO2 Plutonium trickshotting mod menu/custom gun mod setups.

Disclaimer: This tutorial is for custom game trickshotting purposes only. Using these mods in a public match could cause your Plutonium account to be banned.

All original credits belong to the original mod creators and communities. -> https://vapourscripts.com/plutonium & https://x.com/leetinservers

## Included
- Custom guns pack, including guns from various CODs (Leetin server guns)
- Trickshot mod menu (Jiggy V4.3 Vapour)
- Installation screenshots
- Basic setup instructions

## Requirements:
- BO2 Multiplayer working normally first
- Plutonium T6 installed
- WinRAR or 7-Zip

## Installation

1. Create a folder on the desktop called "BO2 Mods".
2. Download the first mod file, the "Jiggy V4.3 Vapour.rar" file from /mod files in this Github repository.
3. Download the second mod file from the MediaFire link: https://www.mediafire.com/folder/45679pfigcfjo/mp_leetin_pub ![MediaFire Download Page](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/e520f8d5ca110b9b421d5d029e0ad4cbce5bc290/screenshots/16%20media%20fire%20link.png?raw=true)
4. Install and launch Plutonium T6 (Black Ops 2 Multiplayer) like normal and go to the main menu at least once (generates t6 folder and files).
5. Press "Win + R" on your keyboard and type "%localappdata%". Click Ok. 

![localappdata](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/e08b2020aedd1097d21aa412910f16f169e7f105/screenshots/3%20localappdata.png?raw=true)

6. Navigate to the "Local" folder. Find "Plutonium", then "storage", then T6. This is the folder where you'll be putting the mods.
   Inside of the t6 folder, create a new folder called "mods" if one doesn't already exist. Make sure it starts with a lowercase "m". ![Plutonium folder](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/40920136aca27acd56a07c546391c28d4c6ff468/screenshots/4%20Plutonium%20folder.png?raw=true) ![Plutonium storage](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/40920136aca27acd56a07c546391c28d4c6ff468/screenshots/5%20Plutonium%20storage.png?raw=true) ![storage t6](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/40920136aca27acd56a07c546391c28d4c6ff468/screenshots/6%20storage%20t6.png?raw=true) ![t6 without maps](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/62eefee28f65a3d25234cb90e003e61d6cc4f6c0/screenshots/17%20t6%20without%20maps.png?raw=true)
7. Back in the "BO2 Mods" folder on the desktop, open up the extracted "Jiggy V4.3" folder and copy the entire "maps" folder into the Plutonium\storage\t6 folder. ![bo2 mods folder](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/70f5851d618a7e14397d17639ea075e7602bf9da/screenshots/bo2%20mods%20folder.png?raw=true) ![jiggy folder](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/c8064b4e63bb23d82a3b94fc7093ff573ad2b33e/screenshots/jiggy%20folder.png?raw=true) ![t6 with maps](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/62eefee28f65a3d25234cb90e003e61d6cc4f6c0/screenshots/7%20t6%20with%20maps.jpg?raw=true)
8. Now navigate to the "mods" folder inside of Plutonium/storage/t6, and paste the extracted "mp_leetin_pub" folder into it. ![mods leetin folder](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/18%20mods%20leetin%20folder.png?raw=true)
9. The mod files are now loaded into the game's Plutonium folder. Load the game like normal. ![launch plutonium](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/8%20launch%20plutonium.png?raw=true)
10. On BO2's main menu, select "ONLINE", then "MODS" and you should see the newly installed "@leetinservers". Load it. ![Main Menu BO2](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/9%20Main%20Menu%20BO2jpg.jpg?raw=true) ![leetin mod](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/10%20leetin%20mod.jpg?raw=true)
11. You should immediately notice the Leetin servers overlay for the game after the mod is loaded, which changes the background and makes the text purple. Try going to "Create a Class" and navigate through the various weapons to find the mod's custom weapons are loading. ![mod main menu](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/12%20mod%20main%20menu.jpg?raw=true) ![check custom classes](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/3fbecf0eecc6b68c2777c703e1ab90238155a5a9/screenshots/13%20check%20custom%20classes.jpg) 
12. The first time you load the mod, your account's progess may reset. This should only happen when the mod is initially loaded. If you want max prestige again, open up plutonium's console command window by typing ` and type "unlockall". Hit enter. ![unlockall](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/89b359f18d79d34bc3a5a6297c668d04e9ad7de1/screenshots/19%20unlockall.png?raw=true)
13. Launch a custom game. When the game starts, select a class and you'll see the mod menu's custom introduction message. Hit Aim + Knife to open the Mod Menu.  ![mod menu comes up](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/89b359f18d79d34bc3a5a6297c668d04e9ad7de1/screenshots/14%20mod%20menu%20comes%20up.jpg?raw=true) ![aim knife for menu](https://github.com/Rainy-City1212/BO2-Plutonium-Trickshot-Mods-Guide/blob/89b359f18d79d34bc3a5a6297c668d04e9ad7de1/screenshots/15%20aim%20knife%20for%20menu.jpg?raw=true)
14. You now have a trickshotting mod menu and custom guns from various CoD games. Enjoy.
